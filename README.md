# Jobready
 A platform where employers can find employees and workers can showcase their services.

 📖 Overview
Jobready is a simple web application that connects employers with workers.  
Employers can browse service packages and hire talent, while workers can create packages and track their earnings.

 **Features**
- **Landing Page & Portal**: Login and registration system
- **Employer Dashboard**: Browse and purchase service packages
- **Worker Dashboard**: Create packages and monitor earnings
- **Authentication**: Mock login/signup switching
- **Responsive Styling**: Centralized CSS for layouts and colors

---

## 🛠 Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Authentication Logic:** Vanilla JS (`auth.js`)
- **Main Logic:** Vanilla JS (`app.js`)
- **Version Control:** Git & GitHub
